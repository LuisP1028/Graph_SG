Link Prediction vs Completion

Goal: Differentiate between predicting future and missing graph data.

The difference between link prediction and completion is that the first is a workflow to predict future links While the completion deals with predicting missing links

Knowledge Graph Embedding Models

Unlike node embedding models, knowledge graph embedding models encode both nodes as well as relationships in the embedding space.

Triple

Knowledge graph embedding models use triples to describe graphs. A triple consists of two nodes known as head (h) and tail (t), and a labeled directed relationship (r).

TransE Embedding Model [SIMPLEST BASELINE]
  • Logic: It captures logical chains well, but struggles with distinguishing entities.
  • Mechanism: TransE treats relationships as translations in vector space. Each node (head/tail) and relationship is assigned a coordinate.
  • Proximity: For a true statement like (head, relationship, tail), the model learns to place them so that moving from the head by the relationship vector lands you near the tail.
You must add the specific edge vector to the head to reach the tail: h + r = ~ t
